## 과제 요구사항
- `upDown.js` — 1~50 무작위 숫자, prompt 입력, 맞힐 때까지 반복, Up!/Down! 안내,
  정답 시 시도 횟수 알림, index에 실행 버튼
- `grade.js` — 3과목 배열, prompt 연속 입력, 평균 계산, 60점 기준 합격 여부와 등급 알림
- `bag.js` — showMyBag(), 소지품 객체(이름·수량) 배열 myBag을 반복 출력

## 완료 내용
- 세 프로그램 모두 스펙 그대로 구현, index에 실행 버튼 연결
