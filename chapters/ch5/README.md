## 과제 요구사항
- 미션1 — `/css/style.css` 생성, 전체 글꼴·줄간격·색·배경, h1/h2 강조, 링크 스타일,
  모든 HTML에서 style.css 적용
- 미션2 — 박스 모델: 컨테이너 정렬, 여행 리뷰 카드, 시간표 테이블 꾸미기
- 미션3 — 회원가입 폼 스타일링(입력창·fieldset·버튼)

## 완료 내용
